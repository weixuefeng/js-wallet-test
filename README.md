# js-wallet-test

This repo is wallet test for some cryptcurrencies, just like evm, sei, sui, tron, aptos, etc.

## how to use.

- evm:
```
node evm.js
```

- sei
```
node sei.js
```

- sui
```
node sui.js
```

## how to contribute

You can add new file `${symble}.toLowerCase()`.js, and execute for node xx.js, Please submit pull request.
